# 禁漫天堂(重构) — Venera 漫画源插件

基于 Venera App 的禁漫天堂 (JMComic) 漫画源，支持浏览、搜索、签到、图片分流测速等功能。

> ⚠️ 由于架构限制已移除网络收藏功能

## 功能特性

### 浏览与发现
- 首页推荐（多分区展示）
- 分类浏览：每週必看、成人A漫、主題A漫、角色扮演、特殊PLAY
- 排行榜：总排行 / 月排行 / 周排行 / 日排行
- 关键词搜索（支持多种排序方式）

### 漫画阅读
- 自动解密图片（支持多种混淆算法）
- 缩略图与阅读图分流加载
- 点赞 / 取消点赞
- 评论浏览与发送
- 标签点击直达搜索

### 账号功能
- 账号自动重新登录（可选）
- 每日自动签到（可选）
- 手动签到

### 域名管理
- 启动时自动刷新 API 域名列表（可选）
- 4 条内置备用域名
- 手动刷新域名
- 图片分流切换（1-4 线路）

### 图片分流测速（v1.7.0 新增）
- 一键并行测试 4 条图片 CDN 线路
- 基于真实漫画图片下载测速（3 张图/线路）
- 自动识别并跳过重复 CDN 域名
- 单图 10 秒超时，避免卡死
- 支持重新测速

### 节点延迟测试（v1.7.0 新增）
- 一键测试 4 条 API 节点响应延迟
- 按延迟升序排列，标注最快节点
- 支持重新测速
- 5 秒超时

## 设置项

| 设置项 | 类型 | 说明 |
|--------|------|------|
| 刷新域名列表 | 按钮 | 手动从 CDN 拉取最新域名 |
| 启动时刷新域名列表 | 开关 | App 启动时自动刷新 |
| Api域名 | 下拉 | 选择 API 线路 (1-4) |
| 图片分流 | 下拉 | 选择图片 CDN 线路 (1-4) |
| 节点优选 | 按钮 | 测试 API 节点延迟 |
| 图片分流测速 | 按钮 | 并行测试 4 条图片 CDN 速度 |
| 每日自动签到 | 开关 | 启动时自动签到 |
| 手动签到 | 按钮 | 立即签到 |

## 技术实现

- **数据加密**：AES-ECB 解密 API 响应
- **图片解密**：基于章节 ID 和图片名的 MD5 哈希动态计算混淆参数
- **鉴权**：基于时间戳的 Token 签名
- **域名分发**：从 CDN 拉取加密域名列表，本地解密
