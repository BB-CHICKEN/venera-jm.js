# 禁漫天堂(重构) — Venera 漫画源插件

基于 Venera App 的禁漫天堂 (JMComic) 漫画源，支持浏览、搜索、收藏、签到、节点测速等功能。

## 基本信息

| 项目 | 值 |
|------|-----|
| 名称 | 禁漫天堂(重构) |
| 标识 | `jm` |
| 版本 | `1.6.0` |
| 最低 App 版本 | `1.5.0` |
| 更新地址 | `https://ghfast.top/https://raw.githubusercontent.com/BB-CHICKEN/venera-jm.js/main/recode-jm.js` |

## 功能特性

### 浏览与发现
- 首页推荐（多分区展示）    
- 分类浏览：每週必看、成人A漫、主題A漫、角色扮演、特殊PLAY
- 排行榜：总排行 / 月排行 / 周排行 / 日排行
- 关键词搜索（支持多种排序方式）

### 漫画阅读
- 自动解密图片（支持多种混淆算法）
- 缩略图与阅读图分流加载
- 点赞 / 取消点赞
- 评论浏览与发送
- 标签点击直达搜索

### 收藏管理
- 多文件夹收藏
- 创建 / 删除收藏夹
- 按添加时间 / 更新时间排序

### 账号功能
- 账号登录 / 登出
- 每日自动签到（可选）
- 手动签到

### 域名管理
- 启动时自动刷新 API 域名列表（可选）
- 4 条内置备用域名
- 手动刷新域名
- 图片分流切换（1-4 线路）

### 节点测速（v1.4.0 新增）
- 一键测试所有 API 节点
- 显示 **延迟（ms）** + **下载速度（MB/s）** + **数据量**
- 基于真实内容下载测速（非 ICMP ping）
- 支持重新测速

## 设置项

| 设置项 | 类型 | 说明 |
|--------|------|------|
| 刷新域名列表 | 按钮 | 手动从 CDN 拉取最新域名 |
| 启动时刷新域名列表 | 开关 | App 启动时自动刷新 |
| Api域名 | 下拉 | 选择 API 线路 (1-4) |
| 图片分流 | 下拉 | 选择图片 CDN 线路 (1-4) |
| 收藏夹排序 | 下拉 | 添加时间 / 更新时间 |
| 每日自动签到 | 开关 | 启动时自动签到 |
| 手动签到 | 按钮 | 立即签到 |
| 节点测速 | 按钮 | 测试所有节点速度 |

## 技术实现

- **数据加密**：AES-ECB 解密 API 响应
- **图片解密**：基于章节 ID 和图片名的 MD5 哈希动态计算混淆参数
- **鉴权**：基于时间戳的 Token 签名
- **域名分发**：从 CDN 拉取加密域名列表，本地解密
